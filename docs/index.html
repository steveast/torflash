⚡ TorFlash

Search torrents on Rutor, NoNaMe-Club & RuTracker. Download via libtorrent. Auto-copy to USB flash drive with smart FAT32 splitting.

Download Binary AppImage GitHub

Multi-source search

Rutor (mirror fallback), NoNaMe-Club, RuTracker with login & proxy. Category filters, history, autocomplete.

Smart flash copy

Auto-detect USB at /run/media. MKV-aware splitting via mkvmerge for FAT32. Safe eject with udisksctl.

Library & seeding

Persistent library in ~/Storage. Resume data restored on startup. Pause, resume, re-check per torrent.

Rich details

Posters, screenshots gallery, mediainfo (codec, resolution, audio), live speed graph, daily stats.

Themes & i18n

Light, dark, and auto themes. Russian and English interface with language switcher.

Self-update

Checks GitHub Releases automatically. One-click update with self-restart. CLI mode for headless servers.

Quick install

mkdir -p ~/Apps/TorFlash && cd ~/Apps/TorFlash
curl -L -o TorFlash https://github.com/steveast/torflash/releases/latest/download/TorFlash
chmod +x TorFlash
./TorFlash
